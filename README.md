# 技术博客存档
博客地址: [http://liucaiquan.site/about](http://liucaiquan.site/about)

不同设备编辑：

	#git clone git@github.com:CaiquanLiu/Blog.git
	#cd Blog
	#npm install -g hexo-cli
	#npm install
	#npm install hexo-deployer-git

注意
* 不需要hexo init这条指令
* 忽略安装过程中fevent相关的警告
* Mac/Windows/Linux环境均适用
